# 🥊 Luffy vs Imu: Stickman Battle

An AI-powered, browser-based fighting game featuring the epic showdown between **Monkey D. Luffy** and the mysterious **Imu**. Built as a demonstration of rapid game prototyping using modern AI orchestration.

**🎮 [Play the Game Here](https://rehaman07.github.io/Stickman/)**

---

## 🚀 Overview

This project was built to explore the intersection of web technologies and AI-assisted development. By leveraging AI for logic generation, physics debugging, and asset conceptualization, I was able to transition from a design concept to a playable GitHub Pages deployment in record time.

This project is part of my submission for the **Google I/O 2026: Code the Countdown** showcase.

## ✨ Features

- **Iconic Characters:** Play as Luffy (Gear 5 style) against the formidable Imu.
- **Dynamic Combat:** Smooth animations and collision detection powered by custom JavaScript logic.
- **Immersive Audio:** Features the "Drums of Liberation" to enhance the gameplay experience.
- **Responsive Web Design:** Playable across different browser environments.
- **AI-Enhanced Dev:** Core game mechanics and character behaviors were optimized using LLM-guided coding.

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3
- **Scripting:** Pure JavaScript (ES6+)
- **Hosting:** GitHub Pages
- **AI Tools:** Google Antigravity 

## 🕹️ How to Play

1.  **Open the Game:** Visit the [Live Demo](https://rehaman07.github.io/Stickman/).
2.  **Controls:** * Use the on-screen buttons or keyboard (if mapped) to move and attack.
    * Defeat Imu before your health reaches zero!
3.  **Goal:** Experience the "Drums of Liberation" as you fight the final boss of the World Government.
